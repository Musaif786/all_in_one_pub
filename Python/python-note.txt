How to create venv on windows Using cmd:
steps:
	python -m venv venv
	cd /venv/Script/
	activate
here you can directly see ur in venv

steps:
	python -m venv venv
	cd /venv/Script/
	./activate.bat
here you can't see ur in venv but while executing python file then you will see which executable file is using
